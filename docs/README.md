# Documentación — Plataforma de Centros de Acopio (Venezuela)

Proyecto **sin fines de lucro** que conecta a la ciudadanía con los centros de acopio
activos tras el terremoto del 24 de junio de 2026, y da a cada centro una herramienta
mínima para registrar lo que recibe.
